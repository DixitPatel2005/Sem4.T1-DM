Q1 Marks: 1
If A = {1,2,3,4,6} and B = {2,3,4}, which one of the following is correct?
Show Answer

Final Answer: B is a subset of A
Q2Marks: 1
What is the value of (A − B) ∪ (B − A) ∪ (A ∩ B)?
Show Answer

(A − B) contains elements only in A.

(B − A) contains elements only in B.

(A ∩ B) contains common elements.

Together, these three parts cover all elements belonging to A or B.

Final Answer: A ∪ B

Q5Marks: 1
If X ∪ Y has 40 elements, X has 28 elements and Y has 22 elements, how many elements does X ∩ Y have?
Show Answer

Use the formula:

n(X ∪ Y) = n(X) + n(Y) − n(X ∩ Y)

40 = 28 + 22 − n(X ∩ Y)

40 = 50 − n(X ∩ Y)

n(X ∩ Y) = 10

Final Answer: 10
Q6Marks: 1
If set A contains 3 elements and set B contains 6 elements, what is the minimum number of elements that (A ∪ B) can have?
Show Answer

The union is minimum when all elements of A are already in B.

Then A ⊆ B, so A ∪ B = B.

Thus minimum number of elements = 6.

Final Answer: 6
